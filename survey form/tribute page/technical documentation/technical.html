Introduction

JavaScript is a programming language that adds interactivity to websites. It is widely used for web development.

You can write JavaScript inside <script> tags in HTML.

Variables

Variables store data values. In JavaScript, you can declare variables using var, let, or const.

let name = "John";
Functions

Functions are blocks of code designed to perform a task.

function greet() {
  console.log("Hello!");
}
Loops

Loops are used to repeat actions multiple times.

for(let i=0; i<5; i++){
  console.log(i);
}
Conclusion

JavaScript is essential for creating dynamic and interactive websites. Mastering it opens doors to advanced web development.
